# Tourism Management System (TMS)

A web-based Tourism Management System built using PHP and MySQL. This project helps users explore tour packages, make bookings, and manage travel information easily through a simple and user-friendly interface.

---

## Features

- User Registration & Login
- Tour Package Listing
- Package Details View
- Online Booking System
- Booking History
- User Profile Management
- Enquiry System
- Admin Dashboard
- Package Management
- User Management

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Installation Guide

### Step 1: Install XAMPP

Install XAMPP and start:

- Apache
- MySQL

---

### Step 2: Move Project Folder

Copy the project folder into:

```bash
C:\xampp\htdocs\
```

Example:

```bash
C:\xampp\htdocs\tms
```

---

### Step 3: Create Database

1. Open phpMyAdmin
2. Create a new database
3. Import the SQL file into the database

---

### Step 4: Configure Database

Open:

```bash
includes/config.php
```

Update database credentials if needed.

---

## Run the Project

Start Apache and MySQL from XAMPP Control Panel.

Then open your browser and visit:

```bash
http://localhost/tms
```

---

